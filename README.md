# ProjectFlow - Project Management System

ProjectFlow is a modern, fast, and intuitive project management tool designed to help teams collaborate seamlessly. Built with a focus on speed and aesthetics, it reduces the friction of managing complex workflows.

-   **Modern Dashboard**: Get a bird's-eye view of all your projects and their health.
-   **Task Management**: Create, assign, and track tasks with status indicators (To Do, In Progress, Completed).
-   **Project Organization**: Group tasks into projects for better clarity.
-   **Team Collaboration**: Assign tasks to team members and track accountability.

## üì¶ Installation

To get a local copy up and running follow these simple steps.

### Prerequisites

*   npm
    ```sh
    npm install npm@latest -g
    ```
*   MongoDB running locally or a cloud connection string.

### Setup

1.  **Clone the repo**
    ```sh
    git clone https://github.com/your_username/project-management-system.git
    cd project-management-system
    ```

2.  **Backend Setup**
    ```sh
    cd backend
    npm install
    # Create .env file and add your config (PORT, MONGO_URI, etc.)
    npm run dev
    ```

3.  **Frontend Setup**
    ```sh
    cd frontend
    npm install
    npm run dev
    ```

4.  **Access the App**
    Open your browser and navigate to `http://localhost:5173`.
